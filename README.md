#Shorten.io - Modern URL Shortener
A sleek, fast, and user-friendly URL shortener built with Flask and TailwindCSS.

#Overview
Shorten.io is a modern URL shortener that allows users to convert long URLs into compact, shareable links. Built with Flask for the backend and TailwindCSS for a sleek UI, it supports single-page operation with instant link generation, no page reloads, and clickable results.

#Features
1.Shorten URLs instantly with a single click
2.Beautiful responsive UI (Light & Dark mode)
3.Clickable short URLs
4.Prevents collisions for unique links
5.Fast and lightweight with in-memory storage (can be upgraded to a database)
6.Works entirely without page reloads using AJAX (fetch)

#Tech Stack
Backend: Python 3, Flask
Frontend: HTML, TailwindCSS, JavaScript
Others: Random module for generating unique short codes
